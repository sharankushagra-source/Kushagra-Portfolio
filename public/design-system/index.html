Handoff · documentation

Developer handoff

How the system is documented and handed to engineering: link one stylesheet and the bundle, then compose with the components — every token, spec and state already defined, so there's no accessibility audit bolted on at the end. (AccessibleFoundations below is an illustrative alias for the bundle's namespace.)

// link the system
<link rel="stylesheet" href="styles.css" />
<script src="_ds_bundle.js"></script>

// read components off the namespace, then compose
const { Button, Input, Toggle, Card } = window.AccessibleFoundations;
<Button variant="primary">Create New</Button>
